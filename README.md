# ICGroup Frontend

Фронтовая часть проекта ICGroup.  
Здесь будет разрабатываться клиентское приложение на Next.js (App Router, TypeScript).

## Технологии

- Next.js
- React
- TypeScript
- ESLint + Prettier
- Husky (pre-commit)
- Jest + React Testing Library

## Назначение

Этот репозиторий содержит публичный сайт ICGroup, включая:

- страницы услуг
- динамические маршруты
- интеграцию с backend API
- авторизацию через NextAuth
- SSR/SSG/ISR

## Статус

Проект находится на этапе начальной настройки.
